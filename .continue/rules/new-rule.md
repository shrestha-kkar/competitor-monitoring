# Global AI Instructions

Before making any implementation decision, always read:

* .github/copilot-instructions.md

Rules:

* Follow existing architecture strictly
* Reuse existing components/services/utilities
* Avoid duplicate implementations
* Use TypeScript strictly
* Avoid `any`
* Keep components modular and reusable
* Do not modify unrelated code
* Explain implementation plan before coding
* Follow existing coding patterns in the repository
* Prefer maintainable and readable code over clever code

When debugging:

* Identify root cause first
* Do not patch blindly
* Explain why the issue occurred before fixing
